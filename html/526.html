Mathematics for economics

This course is ongoing, and material will be modified throughout. Registered students can find problem sets, solutions and grades on UBC Connect. Other users interested in problem sets and solutions should contact me. TeX code for all documents is available from a git repository. If you want to know when I have made changes to the notes, you can sign up as a "Watcher" on bitbucket, or subscribe to the repository rss feed. Please contact me if you notice any errors or want to suggest changes or additions.
